# Ejecutar dev
1. Clonar repositorio
2. Instalar dependencias con pnpm 
```
pnpm install
```
3. Tener [Nest Cli](https://www.npmjs.com/package/@nestjs/cli) instalado
4. Levantar la BD con Docker
```
docker-compose up -d
```

## Stack usado
* MongoDB
* Docker
* Nest
